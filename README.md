# 技能随机分配系统 - 使用说明

## 项目简介

这是一个基于HarmonyOS ArkTS开发的技能随机分配系统，可以帮助用户将一组技能随机分配给多个玩家。系统支持预设技能库和自定义技能两种模式，适用于游戏技能分配、任务分配、团队分工等场景。

## 功能特性

### 🎯 核心功能
- **预设模式**：使用内置的技能库进行快速分配
- **自定义模式**：完全自定义技能名称、描述和类型
- **随机分配**：公平、随机的技能分配算法
- **结果展示**：清晰展示每个玩家分配到的技能

### 📱 页面结构
1. **首页** - 选择分配模式
2. **预设模式** - 从预设技能库选择
3. **自定义模式设置** - 设置技能数量和分配数量
4. **自定义技能输入** - 输入自定义技能信息
5. **分配结果** - 展示随机分配结果

## 快速开始

### 1. 启动应用
应用启动后进入首页，可以看到两个主要选项：
- **预设模式**：使用系统预设的技能库
- **自定义模式**：创建自己的技能列表

### 2. 预设模式使用流程

#### 步骤1：选择预设模式
点击首页的"预设模式"按钮进入预设技能库页面。

#### 步骤2：设置玩家数量
- 使用滑块选择玩家数量（1-10人）
- 系统会自动计算每人分配的技能数

#### 步骤3：开始分配
点击"开始分配技能"按钮，系统将：
1. 从预设技能库中随机选择技能
2. 平均分配给每个玩家
3. 显示分配结果

#### 步骤4：查看结果
在结果页面可以看到：
- 每个玩家分配到的技能列表
- 技能类型标识（主动/被动/大招/特殊）
- 技能详细描述
- 重新分配按钮

### 3. 自定义模式使用流程

#### 步骤1：选择自定义模式
点击首页的"自定义模式"按钮进入设置页面。

#### 步骤2：设置技能参数
- **总技能数**：输入需要创建的技能总数（1-50）
- **每人分配数**：输入每个玩家分配的技能数
- 系统会验证：每人分配数 ≤ 总技能数

#### 步骤3：输入技能信息
点击"下一步"进入技能输入页面：
1. **技能名称**：为每个技能命名
2. **技能描述**：详细描述技能效果（最多3行）
3. **技能类型**：左右滑动选择技能类型
   - 🔵 主动技能
   - 🟢 被动技能
   - 🟠 大招技能
   - 🟣 特殊技能

**输入提示**：
- 技能名称和描述不能为空
- 输入时焦点稳定，不会自动跳转
- 技能描述支持多行输入（最多3行）
- 技能类型选择器支持左右滑动

#### 步骤4：开始分配
点击"开始分配技能"按钮，系统将：
1. 验证所有技能信息是否完整
2. 随机打乱技能顺序
3. 平均分配给每个玩家
4. 跳转到结果页面

#### 步骤5：查看自定义结果
在结果页面可以看到：
- 每个玩家的技能分配情况
- 自定义的技能名称和描述
- 技能类型颜色标识
- 重新分配或返回首页选项

## 详细功能说明

### 预设技能库
系统内置了丰富的技能库，包含：
- **攻击类技能**：各种伤害技能
- **防御类技能**：保护、治疗技能
- **辅助类技能**：增益、控制技能
- **特殊技能**：独特效果技能

### 技能类型说明
1. **主动技能**（蓝色）
   - 需要主动使用的技能
   - 通常有冷却时间
   - 消耗资源（法力、能量等）

2. **被动技能**（绿色）
   - 自动生效的技能
   - 常驻效果或触发效果
   - 不需要主动使用

3. **大招技能**（橙色）
   - 强力的终极技能
   - 通常有较长冷却时间
   - 对战局有重大影响

4. **特殊技能**（紫色）
   - 独特的机制技能
   - 可能改变游戏规则
   - 稀有或限定技能

### 分配算法
系统使用公平的随机分配算法：
1. **随机打乱**：所有技能随机排序
2. **平均分配**：按顺序分配给每个玩家
3. **平衡性**：确保每个玩家获得相同数量的技能
4. **随机性**：每次分配结果都不同

## 使用技巧

### 输入优化
1. **快速输入**：在自定义模式中，可以连续输入而不丢失焦点
2. **批量操作**：先填写所有技能名称，再填写描述
3. **类型选择**：左右滑动选择器查看所有类型

### 验证提示
- 红色边框表示必填项未填写
- Toast提示会指出具体哪个技能需要完善
- 系统会自动滚动到需要完善的技能位置

### 结果查看
- 点击技能卡片可以查看详细描述
- 使用颜色快速识别技能类型
- 支持重新分配获取新的结果

## 常见问题

### Q1: 为什么输入时会跳转到下一个输入框？
**A**: 已修复！系统现在使用本地状态管理，输入时不会触发全局重新渲染，焦点保持稳定。

### Q2: 技能描述可以输入多少文字？
**A**: 技能描述最多显示3行，超过部分可以通过滚动查看。

### Q3: 如何修改已输入的技能？
**A**: 在技能输入页面，可以直接点击已填写的输入框进行修改。

### Q4: 分配结果不满意怎么办？
**A**: 可以点击"重新分配"按钮获取新的随机结果。

### Q5: 可以保存自定义的技能库吗？
**A**: 当前版本不支持保存功能，每次需要重新输入。后续版本会考虑添加保存功能。

### Q6: 最多支持多少技能和玩家？
**A**: 
- 预设模式：最多10个玩家
- 自定义模式：最多50个技能，每人分配数不超过总技能数
