# 快速入门指南

## 🚀 5分钟上手

### 第一步：选择模式
1. 打开应用，进入首页
2. 选择分配模式：
   - **预设模式**：快速使用系统内置技能
   - **自定义模式**：创建自己的技能列表

### 第二步：预设模式（快速开始）
1. 点击"预设模式"
2. 拖动滑块设置玩家数量（1-10人）
3. 点击"开始分配技能"
4. 查看分配结果

### 第三步：自定义模式（完全控制）
1. 点击"自定义模式"
2. 设置参数：
   - 总技能数（例如：10）
   - 每人分配数（例如：3）
3. 点击"下一步"
4. 为每个技能输入：
   - 名称（必填）
   - 描述（必填）
   - 类型（左右滑动选择）
5. 点击"开始分配技能"
6. 查看自定义分配结果

## 📱 界面说明

### 首页
```
[预设模式] [自定义模式]
```
- 两个大按钮，清晰明了
- 点击即可进入对应模式

### 预设模式页面
```
玩家数量：[滑块] 5人
每人分配：2个技能
[开始分配技能]
```
- 滑块调整玩家数量
- 自动计算每人技能数
- 一键开始分配

### 自定义模式设置
```
总技能数：[输入框] 10
每人分配数：[输入框] 3
[下一步]
```
- 输入数字参数
- 自动验证合理性
- 下一步进入技能输入

### 技能输入页面
```
技能 1 [主动技能]
技能名称：[输入框]
技能描述：[多行输入框]
技能类型：[◀ 主动 被动 大招 特殊 ▶]
```
- 每个技能一个卡片
- 输入框焦点稳定
- 类型选择器可滑动
- 自动保存输入内容

### 结果页面
```
玩家 1
- 技能A [主动]：描述...
- 技能B [被动]：描述...
- 技能C [大招]：描述...

[重新分配] [返回首页]
```
- 按玩家分组显示
- 颜色标识技能类型
- 支持重新分配

## 🎮 使用场景示例

### 场景1：游戏技能分配
```
玩家数量：4
总技能数：12
每人分配：3个技能
```
- 为4名玩家分配12个游戏技能
- 每人获得3个随机技能
- 确保公平随机

### 场景2：团队任务分配
```
玩家数量：5
总技能数：15
每人分配：3个任务
```
- 将15个任务分配给5个团队成员
- 每人负责3个任务
- 随机分配避免偏袒

### 场景3：学习小组分工
```
玩家数量：6
总技能数：18
每人分配：3个专题
```
- 将18个学习专题分配给6个小组
- 每组研究3个专题
- 随机分配激发兴趣

## 💡 使用技巧

### 输入技巧
1. **快速填写**：先填完所有技能名称，再填描述
2. **类型选择**：左右滑动查看所有选项
3. **焦点保持**：输入时不会跳转，可连续输入

### 验证提示
- ❌ 红色边框：必填项未填
- 📱 Toast提示：具体错误信息
- 🔍 自动定位：滚动到错误位置

### 结果操作
- 🔄 重新分配：获取新的随机结果
- 🏠 返回首页：开始新的分配
- 📋 查看详情：点击技能卡片

## ⚠️ 注意事项

### 参数限制
- 玩家数量：1-10人
- 总技能数：1-50个
- 每人分配数：≤总技能数

### 输入要求
- 技能名称：不能为空
- 技能描述：不能为空
- 技能类型：必须选择

### 性能提示
- 大量技能时（>20），输入页面可能稍慢
- 建议分批输入或使用预设模式
- 结果页面支持流畅滚动
